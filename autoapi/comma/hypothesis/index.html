adjust_cases(daily_positive_cases: pandas.Series) → pandas.Series

Ensures the length of daily_positive_cases matches the given steps.

If the length of daily_positive_cases is less than the provided steps, (this might happen if there are missing data for some days in RIVM) this function will repeat the last row of daily_positive_cases until its length is equal to steps.

Args:

daily_positive_cases (pd.Series): Series of positive cases per day

Returns:

pd.Series: Modified series with length equal to steps

static scale_cases_to_population(daily_positive_cases: pandas.Series, real_size: int, sim_size: int)

We compute the number of new positives in day _i_ for our simulated population as new_cases_sim = (n/N)*M where N is the size of the simulated population, M is the size of the real population, n is the number of new positives reported on RIVM for that day.

Args:

daily_positive_cases(pd.Series): Daily positive cases real_size(int): Size of the original population sim_size(int): Size of the simulated population

Returns:

pd.Series: Daily positive cases based on simulated population size
