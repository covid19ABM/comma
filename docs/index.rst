Welcome to Comma's documentation!
===================================================

`comma` is a Python package to run agent-based simulations for exploring and analysing the mental health outcomes during various lockdown policies.

**There's a lot here, where should I start?**

- take a look at the overview, which provides a general introduction and instructions on how to install comma
- when you feel ready to try it out, the tutorials will help you get started using comma
- Do you have questions? Send us an email at e.viviani@esciencecenter.nl or kristina.thompson@wur.nl. We're happy to help!
